## 2. 확정된 핵심 요건 (내부회의 2026-04-17)

### 2-1. IP 국가 감지 — MaxMind DB 필수 운영

| 항목 | 내용 |
|------|------|
| **방식** | MaxMind GeoIP2 DB 기반 IP→국가 매핑 |
| **업데이트 주기** | 최소 월 1회, 권장 주 2회 |
| **근거** | IP 대역대는 수시 변경됨. 업데이트 미흡 시 국가 오탐률 증가 → 잘못된 플로우 적용 위험 |
| **운영 방안** | MaxMind 구독 → 자동 다운로드 스크립트 → 서버 반영 (무중단) |
| **Fallback** | MaxMind 장애 시 Cloudflare CF-IPCountry 헤더 보조 사용 |

```
[운영 흐름]
MaxMind GeoLite2/GeoIP2 → 주 2회 자동 갱신 → 서버 로드
  └→ 장애 시 Cloudflare CF-IPCountry 헤더 Fallback
```

### 2-2. 기본 언어 — 한국어 고정 + 구글 번역

| 항목 | 내용 |
|------|------|
| **기본 언어** | **한국어 (모든 국가 동일)** |
| **다국어 방식** | 자체 번역 제공 없음. 구글 번역(Google Translate) 위젯 사용 |
| **적용 범위** | 사이트 전체 (랜딩, 가입, 약관, 콘텐츠) |
| **구글 번역 위치** | 상단 헤더 또는 플로팅 버튼 |

> **변경 전**: 국가별 자동 언어 전환 (ko/en/zh-TW/ja/zh-CN)
> **변경 후**: 한국어 고정, 사용자가 필요 시 구글 번역 사용

**참고**: 약관의 경우 구글 번역 정확도에 법적 한계가 있을 수 있음 → 주요 약관은 영문/중문 공식 번역본 별도 마련 권장 (법무팀 검토 사항)

### 2-3. 가입 시 국가 선택 — 쿠키 기반 검증

| 항목 | 내용 |
|------|------|
| **플로우** | 가입 단계에서 사용자가 직접 국가 선택 |
| **검증 방식** | 프론트에서 수집한 쿠키의 국가 정보와 선택한 국가 비교 |
| **불일치 시** | "접속 국가와 선택한 국가가 다릅니다. 다시 선택해주세요." 안내 |
| **Auth 연동** | Auth 서버는 국가 정보 직접 확인 불가 → **프론트 쿠키 국가 정보를 Auth 가입 단계에 전달** |
| **가입 후** | 국가 변경 불가 (고정) |

```
[가입 플로우 — 국가 검증]

1. 사이트 접속 → MaxMind IP 감지 → 쿠키에 국가코드 저장 (예: molep_country=KR)
2. 가입 클릭 → 국가 선택 드롭다운 표시 (IP 감지 국가가 기본 선택)
3. 사용자가 국가 선택
   ├─ 쿠키 국가와 일치 → 정상 진행
   └─ 쿠키 국가와 불일치 → "접속 국가와 다릅니다. 다시 선택해주세요." 경고
      └─ 재선택 유도 (강제 차단은 아님? → 확인 필요)
4. 선택된 국가 정보를 Auth 서버에 전달 (쿠키 국가 정보 포함)
5. Auth 서버: 프론트에서 전달받은 국가 기준으로 가입 처리
6. 가입 완료 → 회원 DB에 국가코드 저장 (이후 변경 불가)
```

**기술 상세**:
- 쿠키명: `molep_geo_country` (HttpOnly 아님 — 프론트에서 읽어야 함)
- 쿠키 값: ISO 3166-1 alpha-2 국가코드 (KR, US, TW, JP, CN)
- 쿠키 생성 시점: 사이트 최초 접속 시 서버/엣지에서 설정
- Auth 전달 방식: 가입 API 요청 body에 `detected_country`(쿠키)와 `selected_country`(사용자 선택) 모두 포함

## 5. 기술 요건 요약

### 필수 (내부회의 확정)

| # | 요건 | 상세 |
|---|------|------|
| R-01 | MaxMind GeoIP2 DB 운영 | 주 2회 자동 갱신, 무중단 반영 |
| R-02 | 기본 언어 한국어 고정 | 자체 다국어 없음, 구글 번역 위젯 |
| R-03 | 가입 시 국가 선택 + 쿠키 검증 | 프론트 쿠키 ↔ 선택 국가 비교, 불일치 시 재선택 유도 |
| R-04 | 쿠키 국가 → Auth 전달 | Auth에서 국가 직접 확인 불가, 프론트 쿠키 정보 참조 |
| R-05 | 가입 후 국가 변경 불가 | DB에 국가코드 고정 저장 |
